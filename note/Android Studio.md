## 快捷键

- 分屏

split vertically或者split Horizontally

- Hierarchy

ctrl+H

- 大纲

Command+Fn+F12

- 搜索

| 操作   | Mac    | Windows     |
| :-------------: | :-------------: | :-------------: |
| 搜索类       | 	Command+O       | Ctrl+N       |
| 文件中搜索       | 	Command+F       | Ctrl+F      |
| 搜索文件       | 	Command+shift+O       | Ctrl+shif+N       |
| 全局查找       | 	Command+shift+F       | Ctrl+shift+F       |

Double shift： Search Everywhere
